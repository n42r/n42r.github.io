# The Basin Network Model
In my doctoral work, I proposed a novel *[data architecture](https://en.wikipedia.org/wiki/Data_architecture)* for inter-organizational data sharing and exchange (something in between a data lake and a data mesh but for multiple organizations). 

My approach is *metadata-driven*: data is exchanged via 'metadata packets' (which I call *Offerings*). 

My proposal includes (1) a specification of an exchange format (the Offering), (2) an framework for the identification of Offerings (which overcomes some problems in W3C's URIs/URNs), and (3) a metadata mapping library.
